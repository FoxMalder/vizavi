# Vizavi

## Режимы сборки проекта
Запуск живой сборки на локальном сервере:
``` bash
yarn start
```

Одноразовая сборка:
``` bash
yarn one
```

Прод-вариант сборки _(минификация CSS и JS)_:
``` bash
yarn one
```

Прод-сборка проекта в _*.zip_-архив в корне проекта:
``` bash
yarn zip
```
